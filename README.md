Yearly Savings Calculator:
This program easily calculates the amount that a person will save in 1 year. Takes the user's monthly savings and converts to yearly savings. Presents the result in a clean and readable format.
How to Run: 
Python app.py
# or
python3 app.py
Example interaction:
How much money do you put in savings each month: 12
At this rate, you will have $ 144.0 in your savings in 12 months
